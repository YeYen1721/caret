# Caret

Team wellbeing app concept and public web presentation prototype exploring burnout signals, lightweight care actions, and collaborative recovery cues.

## Role

Designed the mobile product experience and built the web prototype used to present and test the interaction model.

## Stack

Figma, HTML, CSS, JavaScript

## What I Built

- Designed an iOS-style team dashboard with visible wellbeing states.
- Built the public phone-framed web prototype for presentation and walkthrough use.
- Created interactive member cards, notification states, profile and market views, and care-action overlays.
- Prototyped a lightweight reward loop around care gestures and team support.
- Used playful visual states to make team stress easier to notice and act on.
