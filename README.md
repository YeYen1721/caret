# Caret

Team wellbeing prototype exploring burnout signals, lightweight care actions, and collaborative recovery cues.

## Role

Designed and built by Minwook Shin as a mobile product prototype.

## Stack

HTML, CSS, JavaScript, Figma

## What I Built

- Designed a phone-framed team dashboard with visible wellbeing states.
- Built interactive member cards, notification states, profile and market views, and care-action overlays.
- Prototyped a lightweight reward loop around care gestures and team support.
- Used playful visual states to make team stress easier to notice and act on.
